# Taskmanager

**Note: This is a personal side-project.** I still haven't finished it yet, and I just program here whenever I have some free time to code. It is a work in progress!

## About the Project
A modern web application designed to help users organize their day and manage tasks efficiently.

**Frontend:**
* React.js
* Material UI (MUI) for custom component styling
* React Router for seamless page navigation

**Backend:**
* Java
* Spring Boot
* RESTful API Architecture
